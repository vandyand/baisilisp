basilisp.data.xml.tree
=======================

``basilisp.data.xml.tree`` provides the standard ``clojure.data.xml.tree``
import path. ``seq-tree`` is the generic lazy enter/exit/node tree builder;
``flatten-elements`` and ``event-tree`` are the XML-specific event/tree
transformations shared with :lpy:ns:`basilisp.data.xml`.
